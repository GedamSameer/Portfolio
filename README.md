# Sameer Portfolio (React + Vite + Tailwind)

## Quickstart
```bash
npm i
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Deploy to Netlify
- Drag & drop the `dist/` folder after build, or connect GitHub repo and set build command to `npm run build` and publish directory to `dist`.

### Customize
- Edit project links in `src/components/Projects.jsx`
- Replace `public/Sameer_Gedam_Resume.pdf` with your real resume
- Swap profile image in `Hero.jsx`
